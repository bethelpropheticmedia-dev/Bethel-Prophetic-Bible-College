இளங்களை பாடத்திட்டங்கள்

உங்கள் பாடப்பிரிவைத் தேர்ந்தெடுத்து, வாட்ஸ்அப்பில் புத்தகங்களைக் ரெக்குவெஸ்ட் செய்யுங்கள்.

தொகுதி 1- தீர்க்கதரிசியின் அடிப்படைத்தன்மை

தீர்க்கதரிசி என்பவன் யார்? Request
தீர்க்கதரிசனம் என்றால் என்ன ? Request
விசேஷித்த தீர்க்கதரிசிகள் Request
தீர்க்கதரிசன வரத்தை செயல்படுத்தும் முறை Request

தொகுதி 2 -தீர்க்கதரிசன ஊழியத்தின் பல்வேறு கோணங்கள்

ஐந்து வகை ஊழியத்தை புரிந்ததுகொள்ளுதல் Request
அபிஷேகத்தின் ரகசியங்கள் Request
ஜெப வாழ்க்கையின் கோட்பாடு Request

தொகுதி 3 -தீர்க்கதரிசிகளின் பங்கு

தேவா வார்த்தையை பெறுதல் Request
எதிரில்காலத்தை பற்றி அறிவித்தல் Request
வழிகாட்டுதலும் ஆலோசனைபணுதலும் Request
வியாக்யானம் பண்ணுதல் Request

தொகுதி 4-ஆவிக்குரிய அனுபவங்கள்

வரங்களும் பரிசுத்த ஆவியும் Request
தேவனின் ஏழு ஆவிகள் Request
தீர்க்கதரிசியின் எதிரிகள் Request

தொகுதி 5 -தீர்க்கதரிசியின் விசேஷித்த அனுபவங்கள்

நான்கு வகை தீர்க்கதரிசன காற்று Request
எட்டு வகை தீர்க்கதரிசிகள் Request

டிப்ளமோ பாடத்திட்டங்கள்

உங்கள் பாடப்பிரிவைத் தேர்ந்தெடுத்து, வாட்ஸ்அப்பில் புத்தகங்களைக் ரெக்குவெஸ்ட் செய்யுங்கள்.

தொகுதி 1- தலைமைத்துவம்

தலைவன் என்பவன் யார் ? Request
தலைமைத்துவத்தின் வகைகள் Request
தலைவனின் தகுதிகள் Request

தொகுதி 2-சாத்தன் இயல் இரகசியங்கள்

பில்லிசூனியம் Request

தொகுதி 3 -ஆவிக்குரிய யுத்தமுறை

மிக்மாஸ் Request
ஆவிக்குரிய யுத்த ஆயுதங்கள் Request

தொகுதி 4 -போதகர் இயல்

யாத்ராகமத்தின் ௧௨ பாளையங்கள் Request

தொகுதி 5 -வேத இரகசியங்கள்

தானியேல் புத்தகம் Request
வெளிப்படுத்தலின் புத்தகம் Request
